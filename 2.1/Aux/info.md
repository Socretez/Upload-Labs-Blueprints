## Schematic Details

### 1. Back-End Dev - 16 Nodes

Standalone coding schematic focused on producing high-quality Optimization Code to unlock the `Back-End Dev` achievement target (`59` Optimization Points).

#### Back-End Dev Purpose

- Isolated achievement run schematic (separate from `The Works`)
- Optimization-point push and setup runs
- Triple Optimized + Debugged Optimization Code build path

#### Back-End Dev Requirements

- Ratio presets are valid when both `Double Variables` and `Optimized Debugging` are unlocked.
- Ratio presets are also valid when neither is unlocked.
- If only one is unlocked, ratio adjustments are required.
- Suggested Advanced Research in `Coding Infinite` node: `8+`
- Pause to purchase `Optimized Debugging` as soon as you acquire `3` Optimization Points.

### 2. AGI Basic - 32 Nodes

Standalone AGI schematic starter kit (Basic).

#### AGI Basic Purpose

- Isolated AGI Starter Kit schematic (separate from `The Works`)
- 6 Way Network/CPU/GPU split
  - Standard Downloader → Distilator → Trainer → AGI Core

#### AGI Basic Requirements

- Ratio presets are valid for when all nodes are `Optimized` or `Unoptimized`
  - Optimization Points Recommended: 3 (Compression, Learning, and Distillation)
  - Application Points Recommended: 0 (None)

### 3. AGI Intermediate - 29 Nodes

Standalone AGI schematic starter kit (Intermediate).

#### AGI Intermediate Purpose

- Isolated AGI Starter Kit schematic (separate from `The Works`)
- Curated Network/CPU/GPU split
  - Zipped Game Torrent Downloader → Compressor Max → Distilator → Decomposition + Splicer → Trainer → AGI Core
- Generates 2.4x more neurons per cycle, requires more CPU/GPU per Download than the basic kit to account for increased file quantity.

#### AGI Intermediate Requirements

- Ratio presets are valid for when all nodes are `Optimized` or `Unoptimized`
  - Optimization Points Recommended: 8 (Compression, Torrent Browsing, Learning, Distillation, Decompilation, Rip, Scrapping, Splicer)
  - Application Points Recommended: 1 (Splicer)
